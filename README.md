- [Passport.js](https://www.passportjs.org/)

## Installation
```bash
npm install
```

## Running the app
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Building the app
```bash
npm run buld
```
Built app is available in the `dist` folder.

## Test
```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e
```
